Display current template file and directory name on screen for WordPress development.

== Description ==

This plugin helps WordPress developers by showing which template files are being used on the current page.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/current-page-template-viewer/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Current Page Template Viewer screen to configure
